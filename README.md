## APP Citas
### Backend made with technologies:
###### - MongoDB.
###### - Mongoose.
###### - Express.
###### - ES6.

For requests to the server at the end points use POSTMAN.
![image](https://i.ibb.co/BzVjnYs/postman.png/ "https://www.postman.com/").


Installation:
> npm init -y

> npm i express

> npm i mongoose
